Экзамен по курсу «Программирование–1»

Баллы за задачи складываются, после чего делятся на 2.

  1. [1 балл] Напишите класс EqualsAnything, такой что для любого x выполнено EqualsAnything() == x.
  2. [5 баллов] Напишите класс Vect3D трёхмерных векторов, реализующий:
    1. доступ к координатам в виде v.x, v.y, v.z;
    2. итерацию по координатам (например, реализовав методы __len__ и __getitem__);
    3. векторное произведение в виде v * u, где v и u — векторы;
    4. скалярное произведение в виде v.dot(u);
    5. сумму и разность;
    6. операторы +=, -=, *=;
    7. длину v.length().
  3. [3 балла] Напишите функции:
    1. select_even(l), возвращающую список из чётных элементов списка l;
    2. slice_odd(l), возвращающую список, состоящий из элементов списка l, стоящих на нечётных местах;
    3. reverse_even(l), возвращающую список из чётных элементов списка l, идущих в обратном порядке;
    4. keys_by_value(d), возвращающую список ключей словаря d в порядке возрастания соответствующих значений;
    5. duplicates(l), возвращающую множество (set) из значений, встречающихся в списке l не менее 2 раз.
  4. [1 балл] Напишите генератор Fib, выдающий (yield) поочерёдно числа Фибоначчи.
  5. [1 балл] Напишите функцию diff_at(f, x, x0, order=1), возвращающую производную sympy-выражения f по переменной x порядка order в точке x0.
  6. [7 баллов] Напишите класс Employee, реализующий следующие методы и свойства. Все свойства доступны только для чтения.
    1. Employee.__init__(self, page). Здесь page — адрес домашней страницы пользователя на hse.ru.
    2. Свойство Employee.homepage, возвращающее страницу.
    3. Свойство Employee.full_name, возвращающее ФИО.
    4. [*] Свойство Employee.affiliations, возвращающее набор пар (dept, post), где dept — подразделение (тип Department), а post — должность (строка).
    5. Свойства Employee.emails, Employee.phone_numbers.
    6. Свойство Employee.keywords, возвращающее список пар (number, title). Например, пара (14026796, "торические многообразия") означает, что у пользователя отмечено ключевое слово "торические многообразия", а список всех пользователей с этим словом расположен по адресу http://www.hse.ru/org/persons/?intst=14026796.
  7. [5+2 баллов] Напишите класс Department. Все свойства доступны только для чтения.
    1. Конструктор Department.__init__(self, udept) запоминает udept.
    2. Свойство Department.staff, возвращающий список (employee, post), где employee — сотрудник (тип Employee), а post — должность (тип str).
    3. Свойство Department.homepage, возвращающее домашнюю страницу подразделения.
    4. Свойство Department.title — название подразделения.
    5. [*] Метод класса Department.by_title(cls, title) находит подразделение по названию (указание: сначала создайте список всех подразделений).
    6. [*] Метод класса Department.by_homepage(cls, homepage) аналогично предыдущему пункту, но поиск происходит по адресу домашней страницы.
